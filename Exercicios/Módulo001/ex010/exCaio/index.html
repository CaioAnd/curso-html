Trabalhando com links

Usando links Externos

Usamos o <a> para adicionar um link no seu site.

Clique aqui para acessar o site Corrija-me.

Como abrir em uma nova guia?

Para abrir o link desejado em outro site, adicionamos target="_blank" dentro do nosso <a> para abrir em outra guia. Também adicionamos rel="external" para ser comunicado ao navegador que esse link foi adicionado e não tem relação com a página principal.

Clique aqui para acessar o site Corrija-me em outra guia.

Links interno

Com o mesmo comando <a> podemos adicionar novas páginas no nosso site, caso estejam na mesma pasta que o index ou dentro de outras sub-pastas.

Para adicionar um link, digitamos o seguinte:

<a href="pag002" rel="next"> exemplo </a>

Clique aqui para entrar na segunda página.

Para que serve o rel=""?

Usamos o rel para o código se comunicar com o site. Dentro pode colocar algumas opções como next, prev, external e nofollow.

rel="next"
Utilizada para indicar ao navegador que o link se refere a próxima página do site principal;
rel="prev"
Semelhante a anterior, é utilizada para indicar ao navegador que o link se refere a uma página anterior do site principal, estamos voltando páginas para trás;
rel="external"
Utilizada para indicar ao navegador que site do link adicionado não faz parte do site principal, indica que o site do link seria um outro site externo;
rel="nofollow"
Utilizada para links de outros sites, essa rel expecífica faz com que o site principal não interfira no site do link adicionado.

Links para download

Usamos o <a> também para adicionar links para download de arquivos. Veja o exemplo:

<a href="arquivos/texto.txt"> Clique aqui para baixar </a>
