Imagem Flexível

Como funciona?

Existem varios tamanhos de telas que irão acessar os sites criados, como computadores, televisões, celulares, etc. Para não termos problemas com a resolução de uma imagem colocada no site, usamos o comando <picture> para adicionar algumas imagens que serão alternadas de acordo com a resolução do navegador aberto.

Como usar?

Usamos inicialmente o comando <picture> e dentro iremos adicionar as imagens que serão alternadas. Veja o exemplo: 

    <picture>
        <source media="(max-width: 750px)" srcset="imagens/paisagem-p.png" type="image/png">
        <source media="(max-width: 1050px)" srcset="imagens/paisagem-m.png" type="image/png">
        <img src="imagens/paisagem-g.png" alt="Imagem Dinâmica">
    </picture>

Dentro do comando <source> usamos também:

media
Usamos com o tamanho máximo que a imagem do source irá ficar;
srcset
Usamos com o endereço em que a imagem está armazenada;
type
Usamos com o tipo de imagem que será utilizada (PNG/JPEG).

Inserindo audio

Aonde baixar?

Podemos baixar conteudo em audio pelo Youtube, lá podemos filtrar e encontrar musicas sem direitos autorais.

Como adicionar ao site?

Podemos adicionar aa nossa página da seguinte forma: 

    <audio src="midia/happy-mistake.mp3" controls></audio>

Problemas de compatibilidade

Desta forma apresentada a cima, embora extremamente simples exitem alguns navegadores que apresentam imcompatibilidade com alguns formatos de áudio, por conta disso usamos uma outra forma para conseguir adicionar vários formatos de áudio: 

    <audio preload="metadata" controls>
        <source src="midia/guanacast-33.mp3" type="audio/mpeg">
        <source src="midia/guanacast-33.wav" type="audio/wav">
        <source src="midia/guanacast-33.ogg" type="audio/ogg">
    </audio>

Parâmetros para a tag <audio>

    controls
    Mostra o painel de controle do audio dentro da página
    preload
      auto
      Carrega completamente o audio e todas as suas informações no site, porém pode deixar o site lento;
      metadata
      Carrega apenas os dados do arquivo como nome, tempo de faixa de audio, data, etc;
      none
      Não carrega nada do audio ao abrir o site.
