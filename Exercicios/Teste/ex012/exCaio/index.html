Inserindo vídeo

Aonde baixar?

Podemos baixar vídeos com direitos autorais liberados no site Pexels. Para adicionar o site, usamos a seguinte tag: 

    <video src="midia/Meu-Video.m4v" width="500" controls></video>

Problemas de compatibilidade

Da mesma forma que o audio, esta é a forma mais simples, porém exitem alguns navegadores que apresentam imcompatibilidade com alguns formatos de vídeo, por conta disso usamos uma outra forma para conseguir adicionar vários formatos de vídeo: 

    <video width="500" controls>
        <source src="midia/Meu-Video.m4v" type="video/mp4">
        <source src="midia/Meu-Video.webm" type="video/webm">
        <p>Seu navegador não tem compatibilidade com repução de vídeo</p>
    </video>

Parâmetros para a tag <video>

    width
    Tamanho do vídeo na página
    poster
    Imagem de capa para vídeo
