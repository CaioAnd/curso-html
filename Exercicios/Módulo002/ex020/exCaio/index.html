Trabalhando com Pseudo-classes

Qual sua utilidade?

Pseudo-classe trabalha com o estado que um elemento está (Vazio, pressionada, marcado, etc)

coloque o mouse aqui

podemos adicionar textos escondidos com as pseudo-classes, por exemplo.

Como aplicar no nosso código?

Vamos mostrar como exemplo o div usado anteriormente. Primeiro devemos criar o div com um <p> veja a baixo:

            
    <div>
        coloque o mouse aqui
        <p>
            podemos adicionar textos escondidos com as pseudo-classes, por exemplo.
        </p>
    </div>

Agora vamos alterar exclusivamente os paragrafos dentro apenas do div, para isso usamos "div > p" e neste style vamos usar também o display: none; que faz com que o texto não seja mostrado. Veja melhor no código a baixo:

            
    div > p {
        display: none;
    }

E por ultimo, para mostrarmos o texto apenas quando o mouse estiver em cima da div, usamos nossa pseudo-classe :hover. Vamos colocar div, que será a tag alterada e :hover para criar a função da pseudo-classe. Após isso vamos inserir > p para alterar apenas os paragrafos do div e usar display: block. Veja o código a baixo:

            
    div:hover > p {
        display: block;
        font-size: 0.9em;
        color: rgb(184, 47, 47);
        font-style: italic;
    }

Resumindo

div > p

Usado para alterar os paragrafos dentro de um div

display: none;

Usado para não mostrar texto do conteudo selecionado.

display: block;

Usado para mostrar texto do conteudo selecionado.

:hover

Faz com que o style seja aplicado apenas quando o mouse estiver em cima da tag especificada antes
