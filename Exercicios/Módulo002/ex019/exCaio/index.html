Trabalhando com id e class

Falando sobre id

Qual sua utilidade?

Quando for necessário personalizar tags específicas de uma página, usamos o id para conseguir personalizar individualmente. Um exemplo é o primeiro h1 deste site que está sendo usado um id

Como aplicar no nosso código?

Para aplicar no código, basta colocar o id com o nome desejado na tag que será inserido o identificador, veja o exemplo:

            
    <h1 id="titulo">Trabalhando com id e class</h1>

Para adicionar no style do código, colocamos o nome da tag e depois uma # e o nome do identificador. Depois basta personalizar da forma que desejar:

            
    h1#titulo {
        color: white
    }

Falando sobre class

Qual sua utilidade?

Quando for necessário utilizar mais de uma tag no mesmo id, nós substituímos esse id por um class que é própria para ser utilizada em várias tags em conjunto. Um exemplo são os h2 dessa página que estão cada um de uma cor diferente.

Como aplicar no nosso código?

Para aplicar no código, basta colocar a class com o nome desejado na tag que será inserido o identificador, veja o exemplo:

            
    <h2 class="utilidade">Qual sua utilidade?</h1>

Para adicionar no style do código, colocamos o nome da tag e depois uma # e o nome do identificador. Depois basta personalizar da forma que desejar:

            
    .utilidade {
        color: green
    }
