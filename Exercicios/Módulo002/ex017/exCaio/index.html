Fontes

Qual fontes usar?

Alguns dispositivos como TVs e celulares não tem suporte para todas as fontes. Uma forma de saber quais fontes são seguras é simplesmente pesquisar no google por "CSS web safe font combinations".

Mais de uma fonte selecionada

Podemos também adicionar mais de uma fonte a ser usada para caso não ter a primeira, o navegador irá pegar a segunda e se caso também não suportar a segunda, usar alguma fonte genérica semelhante a escolhida. Usamos dessa forma:

Neste exemplo, a fonte "Arial" é a solicitada. Caso o aparelho não ter essa fonte, ele irá utilizar a fonte "Helvetica". Se caso o dispositivo não ter nenhuma das duas, o navegador irá colocar uma outra fonte da mesma categoria, que no caso seria "sans-serif", uma fonte não serifada.
