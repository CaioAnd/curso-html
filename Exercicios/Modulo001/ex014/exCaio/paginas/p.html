Estilos para <p>

Para mudar a cor do texto, usamos: 

    p{
        color:cor desejada
    }

Para mudar a cor do fundo do texto, usamos: 

    p{
        background-color: cor desejada
    }

Para mudar o tamanho do texto, usamos: 

    p{
        font-size:  númeropx
    }

Para mudar a fonte do texto, usamos: 

    p{
        font-family: nome da fonte
    }
