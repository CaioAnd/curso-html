Paletas de cores e custumização

Paleta de cores

Podemos adicionar cores no nosso site e deixa-lo mais atrativo para os visitantes, porém para obtermos um melhor resultado, precisamos trabalhar com uma paleta de cores. Anotações no caderno mostram como deve ser montada a paleta de cores, usando o circulo cromático.

Como aplicar cores

Para adicionar uma cor ao nosso site, exitem três formas:

  • Representação por código hexadecimal
    •     #0000FF #FFFFFF
  • Representação em RGB
    •     rgb(0,0,255) rgb(255,255,255)
  • Representação por HSL
    •     hsl(0,0%,100%) hsl(240,100%,50%)
Podemos também alterar a transparência da cor dentro do VS code ou adicionando um último elemento na cor.
