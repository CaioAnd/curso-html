Trabalhando com <main>

a tag <main> é utilizada para marcar o conteúdo principal da página, também pode ser utilizado para criar um bloco como o desta página

Comandos para <main>

background-color

Podemos adicionar nesta tag o style background-color para mudar a cor do seu fundo.

border-radios

Podemos arredondar a borda do bloco usando esse style. Também podemos alterar a intensidade usando NUMpx, sendo NUM o número de pixels.

box-shadow

Podemos adicionar uma sombra para nossa caixa, basta configurar a altura, largura e intensidade usando NUMpx, NUMpx, NUMpx e a cor desejada para sombra.

width

Podemos configurar a largura da caixa, basta usar width e o número de pixels desejado.

margin

Podemos centralizar ou colocar a caixa em algum local específico com margin. Para centralizar, usamos margin: auto;

padding

Podemos adicionar bordas para as letras não ficarem muito justas na caixa, basta usar padding e o número de pixels desejados.
