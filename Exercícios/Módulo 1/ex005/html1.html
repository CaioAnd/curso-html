Hierarquia de Títulos

Como funciona a Hierarquia?

Nessa aula aprendemos que cada <h1> não serve apenas como uma letra grande, mas como um título principal, enquanto os outros <h2>, <h3>, <h4>, etc, (até no máximo <h6>) são usados como subtítulos para outros subtemas.

Caso queira ajustar o tamanho da letra do título, é errado usar <h1> como uma fonte maior, nesse caso deve ser ajustado o tamanho da fonte pelas CSS.
