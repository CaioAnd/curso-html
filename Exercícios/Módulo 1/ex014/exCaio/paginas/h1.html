Estilos para <h1>

Para mudar todas as cores do texto, usamos: 

    h1{
        color:cor desejada
    }

Para mudar todas as cores do fundo do texto, usamos: 

    h1{
        background-color: cor desejada
    }

Para mudar todos os tamanhos dos textos, usamos: 

    h1{
        font-size:  númeropx
    }

Para mudar todas as fontes da página, usamos: 

    h1{
        font-family: nome da fonte
    }
