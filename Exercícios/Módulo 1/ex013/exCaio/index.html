Estilos Inline

O que são estilos inline?

Podemos custumizar nossas tags usando style dentro da tag, clique a baixo para ver alguns exemplos:

  • Clique para ver estilos no <body>.
  • Clique para ver estilos no <h1>.
  • Clique para ver estilos no <p>.

Sobre os estilos Inline

Os esilos Inline são utilizados dentro das tags de comando, por isso o nome Inline. Por poluir o código e ser extremamente trabalhoso, não é recomendado usar essa técnica.
